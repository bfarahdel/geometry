Lesson 6-1

Graphing Exponential Functions

Exponential Growth Introduction

An exponential function features a constant base structural value paired with a variable exponent element.

f(x)=bx ,b>1
  • Behavior: Continuous and strictly increasing
  • Domain: All real numbers (-∞,∞)
  • Range: All positive real numbers (y>0)
  • Asymptote: x-axis (y=0)

Example 1: Graphing y = 3x

Graph the function y = 3x. Identify its key metrics.

x -2 -1 0 1 2
y 1/9 1/3 1 3 9

Domain: All Real Numbers (-∞,∞)

Range: Positive Outputs (0,∞)

Transformations Introduction

f(x)=abx-h+k

h-value

Horizontal Shift
Moves right if subtraction (x-h), and left if addition.

k-value

Vertical Shift
Shifts everything up or down. Explicitly changes the horizontal asymptote location.

a-value

Orientation & Scale
Controls vertical scaling stretch, compression, or reflections across the x-axis.

Example 2: Vertical Shift

Graphing: y = 2x + 1

Shifts the baseline parents structural graph of y = 2x vertically up 1 full unit.

  • Asymptote: Displaces upwards to y=1
  • Domain: Unchanged (-∞,∞)
  • Range: Shifts upward with boundary (1,∞)

Exponential Decay Functions

When the base value falls inside fractional limits, the system drops rapidly rather than escalating.

f(x)=bx ,0<b<1
  • Behavior: Continuous and strictly decreasing
  • Domain: All real numbers (-∞,∞)
  • Range: All positive real numbers (y>0)
  • Y-Intercept: Retains standard baseline at (0, 1)

Example 3: Graphing y = (1/3)x

Plot coordinates across inverse powers to clearly map out the structural decay trend line.

Key Coordinates:
(-1, 3),   (0, 1),   (1, 1/3)

  • As x values turn positive, fractional multiplications force outputs closer to zero.
  • The x-axis line (y=0) remains the boundary ceiling limit.
